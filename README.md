## Installation
- **frontend**  
React + Typescript + Vite
```zsh
yarn
```

- **Library and backend**  
```zsh
python3 -m venv venv
source ./venv/bin/activate
pip3 install -r requirements.txt
```

## To Run the project
### Frontend

```zsh
cd frontend
yarn dev
```

### Backend  
```zsh
cd backend
fastapi dev sever.py
```
